# Fundamental Equations:
1. Schrödinger Equation: iħ ∂ψ/∂t = -ħ²/2m ∇²ψ + Vψ
2. Time-Independent SE: Hψ = Eψ
3. Probability Density: P(x) = |ψ(x)|²
4. Normalization: ∫|ψ|² dx = 1
5. Expectation Value: ⟨A⟩ = ∫ψ* A ψ dx
6. Commutation: [x, p] = iħ
7. Uncertainty: Δx Δp ≥ ħ/2
8. De Broglie: λ = h/p
